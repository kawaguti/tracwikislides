tracwikislides by kawaguti

An library for playing presentation slides written by Trac Wiki format

TracWikiSlides is an library for playing presentation slides written by Trac Wiki format. HTML/CSS/JavaScript based. Using jQuery 1.3.1. Slides may work on common browsers such as Safari, FireFox and Google Chrome without any plug-ins. (Japanese) TracWikiSlides は Trac の WIki記法で記述されたの プレゼンテーションスライド を ブラウザ上で実行するライブラリです。 HTML/CSS/JavaScript ベースで、jQuery 1.3.1 を利用しています。 プラグインを使うことなく、Safari, FireFox, Google Chromeなどのブラウザで動作するのではないかと思います。

Dependencies

jQuery 1.3.1
